# 🏇 한국마사회(KRA) 시스템 가이드

## 📋 개요

한국마사회(Korea Racing Authority, KRA)의 경마 시스템, 조직, 운영 체계에 대한 완벽 가이드입니다.

---

## 🏢 한국마사회(KRA) 소개

### 기본 정보

```
명칭: 한국마사회 (Korea Racing Authority)
설립: 1949년
근거 법령: 한국마사회법
주무부처: 농림축산식품부
본부: 경기도 과천시 경마공원대로 107
```

### 주요 역할

1. **경마 시행**

   - 경마 개최 및 운영
   - 경주 기준 수립 및 관리
   - 마권 발매 및 배당금 지급

2. **말산업 육성**

   - 경주마 생산 및 육성
   - 승마 보급 및 저변 확대
   - 말 관련 R&D

3. **공익 사업**
   - 축산 발전 기금
   - 농어촌 특별세
   - 사회공헌 활동

## 🏆 경주마 등급 시스템

### 등급 분류 (1~6등급)

```
1등급: 레이팅 95 이상
- 최상위 경주마
- 상금: 5,000만원~3억원
- 대표 경주: 코리아컵, 대통령배

2등급: 레이팅 85~94
- 상위 경주마
- 상금: 3,000만원~1억원
- 대표 경주: 서울대상, 부산대상

3등급: 레이팅 75~84
- 중상위 경주마
- 상금: 2,000만원~5,000만원

4등급: 레이팅 65~74
- 중위 경주마
- 상금: 1,000만원~3,000만원

5등급: 레이팅 55~64
- 중하위 경주마
- 상금: 500만원~1,500만원

6등급: 레이팅 55 미만
- 하위 경주마
- 상금: 300만원~1,000만원
```

### 레이팅 산정 기준

```
레이팅 = 기본점수 + 가산점 - 감점

기본점수:
- 1등: 10점
- 2등: 7점
- 3등: 5점
- 4등: 3점
- 5등: 1점

가산점:
- 경주 등급에 따른 가산
- 부담중량 고려
- 경주 거리 고려

감점:
- 낮은 등급 경주 출전
- 부진한 성적 지속
```

## 🎫 마권 발매 시스템

### 발매 방식

```
장외발매소:
- 전국 30여개 장외발매소
- 경주 당일 현장 발매

인터넷/모바일:
- 렛츠런파크 홈페이지
- 모바일 앱
- 365일 24시간 발매

전화 발매:
- ARS 시스템
```

### 발매 마감 시간

```
현장/장외:
- 경주 시작 2분 전

인터넷/모바일:
- 경주 시작 1분 전
```

### 구매 한도

```
1회 최소: 100원
1회 최대: 100만원
1일 최대: 무제한 (실명제)
```

## 📊 배당률 시스템

### 배당률 산정 방식

```
파리뮤추얼 방식 (Parimutuel)

배당률 = (총 발매액 - 공제금) ÷ 적중 마권 발매액

공제율:
- 단승식: 17.0%
- 연승식: 20.5%
- 쌍승식: 22.0%
- 복연승식: 24.0%
- 삼복승식: 25.0%
- 삼쌍승식: 27.0%
```

### 최저/최고 배당

```
최저 배당: 1.0배 (원금 회수)
최고 배당: 제한 없음
평균 배당:
- 단승식: 8~12배
- 연승식: 15~30배
- 쌍승식: 50~150배
- 삼쌍승식: 500~2,000배
```
