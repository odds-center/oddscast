##### 4\) 요청/응답 메시지 예제

**요청메시지**

```
http://www.data.go.kr/openapi/service/API160/integratedInfo?ServiceKey=인증키(URL Encode)&numOfRows=10&pageNo=1
```

**응답메시지**

```xml
<?xml version="1.0" encoding="UTF-8" standalone="true"?>
<response>
    <header>
        <resultCode>00</resultCode>
        <resultMsg>NORMAL SERVICE.</resultMsg>
    </header>
    <item>
        <chulNo>1</chulNo>
        <chulNo2>0</chulNo2>
        <chulNo3>0</chulNo3>
        <meet>서울</meet>
        <odds>2</odds>
        <pool>연승식</pool>
        <rcDate>20190601</rcDate>
        <rcNo>1</rcNo>
    </item>
    </body>
</response>
```
