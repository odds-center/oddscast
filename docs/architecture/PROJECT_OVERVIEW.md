# Golden Race - 프로젝트 전체 개요

## 🏇 프로젝트 소개

Golden Race는 경마 애호가들을 위한 종합적인 경마 정보 및 베팅 플랫폼입니다. 모바일 앱과 웹 서버를 통
해 실시간 경주 정보, 베팅 시스템, 그리고 사용자 맞춤형 서비스를 제공합니다.

## 🏗️ 전체 아키텍처

```
goldenrace/
├── mobile/                    # React Native + Expo 앱
│   ├── app/                   # Expo Router 앱 구조
│   ├── components/            # React 컴포넌트
│   ├── lib/                   # API 연동 라이브러리
│   ├── hooks/                 # 커스텀 훅
│   ├── store/                 # 상태 관리
│   └── docs/                  # 모바일 앱 문서
├── server/                    # NestJS 백엔드
│   ├── src/                   # 소스 코드
│   ├── external-apis/         # 외부 API 연동
│   └── docs/                  # 서버 문서
├── docs/                      # 전체 프로젝트 문서 (현재 위치)
└── scripts/                   # 공통 스크립트
```

## 🎯 핵심 기능

### 1. 실시간 경주 정보

- **KRA API 연동**: 한국마사회 공식 API를 통한 실시간 데이터
- **경주계획표**: 예정된 경주 정보 및 일정
- **확정배당율**: 실시간 배당률 정보
- **경주 결과**: 완료된 경주의 상세 결과

### 2. 베팅 시스템

- **베팅 플레이스**: 다양한 베팅 유형 지원
- **베팅 내역**: 사용자별 베팅 기록 관리
- **배당률 계산**: 실시간 배당률 기반 수익 계산
- **베팅 규칙**: 공정하고 투명한 베팅 시스템

### 3. 사용자 서비스

- **소셜 로그인**: Google OAuth 2.0 기반 인증
- **개인화**: 사용자 선호도 기반 맞춤 서비스
- **알림 시스템**: 푸시 알림 및 실시간 업데이트
- **통계 분석**: 개인 베팅 성과 및 경주 분석

## 🔧 기술 스택

### 백엔드 (Server)

- **프레임워크**: NestJS (Node.js)
- **데이터베이스**: MySQL 8.0 + TypeORM
- **인증**: JWT + Passport.js
- **API**: RESTful API + Swagger
- **컨테이너**: Docker + Docker Compose

### 프론트엔드 (Mobile)

- **프레임워크**: React Native + Expo
- **언어**: TypeScript
- **상태 관리**: React Query + Zustand
- **UI 라이브러리**: NativeBase
- **네비게이션**: React Navigation

### 인프라

- **호스팅**: Google Cloud Platform
- **데이터베이스**: Cloud SQL
- **스토리지**: Cloud Storage
- **모니터링**: Cloud Monitoring

## 📈 비즈니스 모델

### 수익 구조

1. **베팅 수수료**: 베팅 금액의 일정 비율
2. **프리미엄 서비스**: 고급 분석 및 통계
3. **광고 수익**: 관련 업체 광고
4. **데이터 라이센싱**: 익명화된 통계 데이터

- **타겟 사용자**: 경마 애호가, 베팅자
- **시장 규모**: 국내 경마 시장 약 3조원
- **경쟁 우위**: 실시간 데이터, 사용자 친화적 UI
- **확장 가능성**: 해외 시장 진출, 스포츠 베팅 확장
