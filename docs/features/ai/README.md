## 🔑 핵심 개념

### LLM 기반 예측

Golden Race는 **별도의 ML 모델 학습 없이** GPT-4, Claude 등의 LLM API를 사용합니다.

#### 장점

✅ **즉시 사용 가능**: API 연동만으로 서비스 시작  
✅ **높은 정확도**: 이미 학습된 모델 활용  
✅ **설명 가능성**: 예측 근거를 자연어로 설명  
✅ **빠른 개선**: 프롬프트 수정으로 즉시 반영

#### 비용

- **GPT-4 Turbo**: 경주 1건 예측 = 약 ₩54
- **캐싱**: 중복 요청 시 비용 1/N
- **구독자 1,000명**: 월 약 ₩1,620,000

## 📊 예측 프로세스

```
1. 사용자 예측 요청 (예측권 1장 사용)
   ↓
2. 경주 데이터 수집 (DB에서 조회)
   - 경주 정보
   - 출전마 정보
   - 과거 성적
   - 배당률
   ↓
3. 프롬프트 생성
   - System Prompt (전문가 역할 정의)
   - User Prompt (경주 데이터 제공)
   ↓
4. LLM API 호출 (GPT-4 또는 Claude)
   ↓
5. 응답 파싱 및 검증
   {
     "firstPlace": "7",
     "secondPlace": "3",
     "thirdPlace": "5",
     "confidence": 85,
     "analysis": "...",
     "warnings": [...]
   }
   ↓
6. 결과 저장 및 캐싱 (Redis, 1시간)
   ↓
7. 사용자에게 결과 반환
```
