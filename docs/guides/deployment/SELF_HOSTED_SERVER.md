## 🏠 자체 서버 운영 시 필요한 것

### 1. 네트워크 설정

```bash
# 공유기 포트 포워딩
80 → 192.168.0.100:80 (HTTP)
443 → 192.168.0.100:443 (HTTPS)
3002 → 192.168.0.100:3002 (API)

# DDNS 설정 (동적 IP 대응)
goldenrace.ddns.net → 집 공인 IP
```

### 2. 보안 설정

```bash
# Firewall (UFW)
sudo ufw allow 22/tcp   # SSH
sudo ufw allow 80/tcp   # HTTP
sudo ufw allow 443/tcp  # HTTPS
sudo ufw enable

# Fail2ban (무차별 대입 공격 방어)
sudo apt install fail2ban
sudo systemctl enable fail2ban

# 자동 보안 업데이트
sudo apt install unattended-upgrades
sudo dpkg-reconfigure --priority=low unattended-upgrades
```

### 3. 백업 시스템

```bash
# 매일 MySQL 백업
0 3 * * * mysqldump -u root -p goldenrace | gzip > /backup/db_$(date +\%Y\%m\%d).sql.gz

# 외부 저장소에 업로드 (S3)
0 4 * * * aws s3 cp /backup/ s3://goldenrace-backup/ --recursive

# 7일 이상 된 백업 삭제
0 5 * * * find /backup -name "*.sql.gz" -mtime +7 -delete
```

### 4. 모니터링

```bash
# Uptime Robot (무료)
- 5분마다 헬스체크
- 다운 시 이메일/SMS 알림

# Prometheus + Grafana
docker run -d -p 9090:9090 prom/prometheus
docker run -d -p 3000:3000 grafana/grafana

# PM2 모니터링
pm2 install pm2-server-monit
```
