# KRA 경마 예상업자 등록 검토

> **작성일:** 2026-04-06
> **상태:** 검토 중 (미결정)
>
> OddsCast의 사업 확장 옵션으로서 한국마사회 경마 예상업자 등록 가능성을 정리한 문서.
> 실행 여부는 법률 자문 후 결정 예정.

---

## 1. 법적 근거

**한국마사회법 제51조 (경마 예상업)**

- 경마 예상 정보를 유상으로 제공하는 사업은 한국마사회의 **허가**를 받아야 함
- 무허가 예상업 영위 시 **2년 이하 징역 또는 2천만원 이하 벌금**
- 온라인 서비스가 "예상업"에 해당하는지는 법률 해석 필요 (별도 자문 권장)

---

## 2. 현재 OddsCast 법적 포지션

| 구분 | 내용 |
|------|------|
| **서비스 형태** | AI 분석 정보 제공 (구독 서비스) |
| **법적 성격** | 정보서비스업 — 주식 투자 뉴스레터와 동일 성격 |
| **별도 허가** | 현재 불필요 (정보서비스업 범주) |
| **주의 사항** | "예상지" 표현 지양 → "AI 분석 리포트", "데이터 분석" 사용 |

---

## 3. 경마 예상업자 등록 요건

| 항목 | 내용 |
|------|------|
| **신청 주체** | 개인 또는 법인 |
| **결격 사유** | 미성년자, 피성년후견인, 금고 이상 형 집행 후 5년 미경과자 |
| **영업 장소** | 경마장 내 또는 KRA 지정 구역 (온라인 허용 여부 별도 확인) |
| **보증금** | 일정 금액 납부 (KRA 규정에 따라 변동) |
| **교육 이수** | KRA 지정 예상업 교육 이수 필수 |

---

## 4. 등록 절차

```
1단계: KRA 예상업 담당부서 문의
         - 서울 / 부산 / 제주 각 경마장 운영팀
         - 고객센터: 1566-3333
         - 공식사이트: kra.co.kr
         - 현재 등록 가능 여부 및 온라인 서비스 적용 기준 확인

2단계: 서류 준비
         - 사업자등록증 (법인의 경우 법인등기부등본)
         - 예상업 영위 계획서
         - 보증금 납부 증명
         - KRA 지정 교육 이수증

3단계: KRA 심사
         - 영업장소 적격 여부
         - 결격 사유 없음 확인
         - 계획서 내용 심사

4단계: 허가증 발급 및 영업 개시
```

---

## 5. 등록 시 기대 효과 (온라인 한정)

> 오프라인(경마장 부스, 단말기 납품)은 사업 방향에서 제외.

| 항목 | 내용 |
|------|------|
| **마케팅** | "KRA 공인 예상 서비스" 표기 가능 → 신뢰도 상승 |
| **법적 안정성** | 예상업 허가로 법적 불확실성 완전 제거 |
| **파트너십** | KRA와 공식 협력 관계 형성 → 데이터 접근성 개선 가능 |

---

## 6. 법률 자문이 필요한 항목

온라인 구독 서비스 특성상 아래 항목은 **경마법 + 정보통신망법** 교차 적용 여부를 전문가와 확인해야 함:

- [ ] 유료 AI 예상 구독이 "예상업"에 법적으로 해당하는지
- [ ] 앱스토어(iOS / Google Play) 출시 시 추가 규제 여부
- [ ] KRA 공공데이터 API 상업적 이용 계약 필요 여부
- [ ] KRA와 데이터 제휴 파트너십 가능성

**추천 채널:** 로톡(lawtalk.co.kr) 또는 헬프미 — 경마/도박법 전문 변호사 1회 상담 (1~3만원 수준)

---

## 7. 단계별 실행 전략

### 단기 (현재 ~ 앱스토어 출시 전)
- 현재 "AI 분석 정보 서비스" 포지션 유지
- 서비스 내 "예상지" 표현 제거 → "AI 분석", "경주 데이터 리포트" 사용
- 법률 자문 1회 진행 (온라인 서비스 해당 여부 명확화)

### 중기 (앱스토어 출시 전후)
- KRA 예상업 등록 또는 협력 파트너십 가능성 문의
- KRA 공공데이터 API 공식 상업적 이용 계약 체결 검토

### 장기 (서비스 성장 후)
- KRA와 온라인 데이터 파트너십 체결
- 경마 외 유사 스포츠(경륜, 경정) 온라인 AI 분석 확장 검토
- ~~오프라인 채널 확장~~ — 제외
