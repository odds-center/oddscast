# Golden Race WebApp

Next.js 반응형 웹앱 (port 3000)

## 빠른 시작

```bash
npm install
cp .env.example .env   # 또는 .env.local
npm run dev
```

**Mock 모드:** `.env`에 `NEXT_PUBLIC_USE_MOCK=true` 설정 시 DB/서버 없이 mock 데이터로 동작합니다.

**상세 가이드:** [docs/guides/WEBAPP_DEVELOPMENT.md](../docs/guides/WEBAPP_DEVELOPMENT.md)
