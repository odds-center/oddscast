## 뉴스레터 운영 가이드

### 발송 일정

| 이벤트 | 발송 시점 | 내용 |
|--------|-----------|------|
| 정기 주간 리포트 | 매주 목요일 10:00 KST | 주말 경주 예고 + AI 주목마 TOP 3 |
| 경주 결과 요약 | 매주 월요일 10:00 KST | 지난 주 결과 리뷰 (선택) |
| 클래식 레이스 특집 | 대형 경주 1주 전 | 코리안더비·대통령배 등 특집 리포트 |
| 가입 웰컴 메일 | 가입 직후 즉시 | 가입 혜택 안내 + 사용 가이드 |

### 성과 측정 지표

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 오픈율 | 25% 이상 | Resend 대시보드 |
| 클릭률 | 5% 이상 | 앱 링크 UTM 파라미터 |
| 구독 전환율 | 비구독자 CTA 클릭 → 구독 | 서버 구독 이벤트 |
| 수신 거부율 | 0.3% 미만 유지 | Resend 수신 거부 트래킹 |

### A/B 테스트 운영

- 매월 1회, 제목 2~3개 버전으로 A/B 테스트
- 수신자 20% 에게 B버전 발송 → 오픈율 높은 버전을 나머지 80%에게 발송
- 결과를 월간 리포트에 기록하고 이후 제목 전략에 반영

### 개인화 우선순위

1. 닉네임 삽입 (기본)
2. 구독 여부에 따른 CTA 분기
3. 주 이용 경마장 기반 경주 정보 우선 표시 (향후)
4. 예측권 잔량 표시 (구독자)
