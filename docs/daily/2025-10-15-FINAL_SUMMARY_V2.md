## 🆕 핵심 추가 기능: AI 예측 업데이트 시스템

### 비즈니스 로직

> **AI 예측을 30분마다 업데이트하여 사용자가 예측권을 반복 사용하도록 유도**

### 수익 증대 효과

```
기존: 1경주당 1장 소비
현재: 1경주당 2~3장 소비 (업데이트마다 재구매)

→ 예측권 소비 2~3배 증가 🎉
→ 추가 구매 유도 💰
```

### 구현 내용

#### 1. 데이터베이스 (1개 컬럼 추가)

```sql
ALTER TABLE prediction_tickets
ADD COLUMN viewed_at DATETIME NULL COMMENT '예측을 본 시점';
```

#### 2. Cron Job (30분마다 업데이트)

```typescript
@Cron('*/30 * * * *') // 30분마다
async updateUpcomingRacePredictions() {
  // 1시간 이내 시작하는 경주만 업데이트
  // 변경사항 있으면 기존 예측 덮어쓰기
  // updatedAt 갱신
}
```

#### 3. API 응답 (업데이트 감지)

```json
{
  "hasViewed": true, // 이미 봤는지
  "isUpdated": true, // 업데이트 되었는지
  "message": "🆕 AI 예측이 업데이트되었습니다!"
}
```

#### 4. 모바일 UI (업데이트 뱃지)

- `PredictionStatusBadge`: 🆕 업데이트됨 / ✅ 확인함 / 🤖 AI 예측
- `AIPredictionPreview`: 업데이트 알림 + 재구매 유도 버튼
