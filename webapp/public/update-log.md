# OddsCast 업데이트 내역

## v2.4.0 — 2026년 3월 19일

### AI 예측 정확도 대폭 개선

- **15요소 분석 모델 (v4)**: 기존 12요소에서 게이트 바이어스, 출전두수 영향, 페이스 전개 시나리오 3개 요소 추가
- **가중치 재조정**: 최근 폼(성적) 비중 상향, 레이팅 과의존 해소, 기수 영향력 강화
- **4단계 분석 프레임워크**: 경주 구조 → 말별 정밀분석 → 승부예측 → 신뢰도 판정
- **경마장별 맞춤 분석**: 서울(잔디/좌회전), 부산(모래/짧은 직선), 제주(소규모) 특성 반영
- **페이스 전개 예측**: 선행마 과다 시 오버페이스 → 추입마 역전 시나리오 자동 분석

### 예측 파이프라인 버그 수정

- 출전 취소마(scratched)가 AI 분석에 포함되던 문제 수정
- 기수 경험 판단 시 말의 경력을 기수 경력으로 오인하던 버그 수정
- 마체중 변화 분석이 특정 데이터 형식에서 작동하지 않던 문제 수정
- 조교사 데이터 조회 범위 확대 (이름 기반 폴백 추가)
- 예측 생성 중 오류 발생 시 영구 잠금 방지
