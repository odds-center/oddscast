#### Savings Plans

- **Compute Savings Plans**: EC2/Lambda/Fargate 할인
- **예상 절감**: 월 **₩40,000** ~ **₩80,000**

#### Auto Scaling 최적화

- **야간 시간대 축소**: 트래픽 적은 시간 인스턴스 축소
- **예상 절감**: 월 **₩20,000** ~ **₩40,000**

- **Redis TTL 연장**: 1시간 → 3시간

- **예상 절감**: API 호출 **70% → 85%** 감소

- 중요 경주 (Grade 1~3): GPT-4 (30%)
