## 4️⃣ 쌍승식 (Exacta)

### 기본 정보

```
선택 마수: 2마리
적중 조건: 1등, 2등을 순서대로 맞춤
난이도: ⭐⭐⭐⭐ (어려움)
평균 배당: 50~150배
공제율: 22.0%
```

### 베팅 방법

#### 단식 (Straight)

```
선택: 5번 → 3번 (순서)
투자: 10,000원

결과:
1등: 5번 ✅
2등: 3번 ✅

배당: 89.5배
환급: 895,000원
수익: 885,000원
```

#### 복식 (Box)

```
선택: 5번, 3번 (순서 무관)
조합: 5→3, 3→5 (2가지)
투자: 5,000원 × 2 = 10,000원

결과:
1등: 5번
2등: 3번
적중: 5→3 ✅

배당: 89.5배
환급: 447,500원 (5,000원 × 89.5)
수익: 437,500원
```

### 조합 계산

```
n마리 선택 시 조합 수 = n × (n-1)

예시:
3마리: 3 × 2 = 6가지
4마리: 4 × 3 = 12가지
5마리: 5 × 4 = 20가지
```

### 전략

```
✅ 유리한 상황:
- 1~2등 예측 가능
- 강력한 1, 2인기마
- 혼전 예상

❌ 불리한 상황:
- 능력 평준화
- 날씨 변수
- 외부 변수 많음
```

```
1등: 3번, 2등: 5번 ✅
1등: 7번, 2등: 5번, 3등: 3번 ✅
1등: 7번, 2등: 3번, 3등: 5번 ✅

배당: 8.5배
환급: 85,000원
수익: 75,000원
```

### 전략

```
✅ 추천 조합:
- 1인기 + 3~5인기
- 2인기 + 4~6인기
- 상위권 확실 + 중위권

❌ 비추천:
- 상위권끼리 (배당 낮음)
- 하위권끼리 (적중률 낮음)
```

## 6️⃣ 삼복승식 (Trifecta Box)

### 기본 정보

```
선택 마수: 3마리
적중 조건: 1등, 2등, 3등 (순서 무관)
난이도: ⭐⭐⭐⭐ (어려움)
평균 배당: 100~500배
공제율: 25.0%
```

### 베팅 방법

```
선택: 5번, 3번, 7번
조합: 6가지 (3×2×1)
- 5-3-7
- 5-7-3
- 3-5-7
- 3-7-5
- 7-5-3
- 7-3-5

투자: 1,000원 × 6 = 6,000원

결과:
1등: 5번
2등: 7번
3등: 3번
적중: 5-7-3 ✅

배당: 285.5배
환급: 285,500원 (1,000원 × 285.5)
수익: 279,500원
```

### 조합 계산

```
n마리 선택 시 조합 수 = n!

3마리: 3×2×1 = 6가지
4마리: 4×3×2 = 24가지
5마리: 5×4×3 = 60가지
6마리: 6×5×4 = 120가지
```

## 7️⃣ 삼쌍승식 (Trifecta)

### 기본 정보

```
선택 마수: 3마리
적중 조건: 1등, 2등, 3등을 순서대로 맞춤
난이도: ⭐⭐⭐⭐⭐ (가장 어려움)
평균 배당: 500~2,000배
최고 배당: 89,000배 (역대 최고)
공제율: 27.0%
```

### 베팅 방법

#### 단식

```
선택: 5번 → 3번 → 7번 (순서)
투자: 10,000원

결과:
1등: 5번 ✅
2등: 3번 ✅
3등: 7번 ✅

배당: 1,285.5배
환급: 12,855,000원
수익: 12,845,000원
```

#### 복식

```
선택:
1등: 5번, 3번 (2마리)
2등: 5번, 3번, 7번 (3마리)
3등: 5번, 3번, 7번, 2번 (4마리)

조합수: 2 × 3 × 4 = 24가지
투자: 1,000원 × 24 = 24,000원
```

### 고배당 전략

```
인기도 분산:
1등: 1~2인기
2등: 4~6인기
3등: 7~10인기

투자 전략:
- 소액 다조합
- 핵심 축마 설정
- 유동적 조합
```

## 💡 Golden Race 개발 시 고려사항

### API 구현

```typescript
// 베팅 타입
enum BetType {
  WIN = 'WIN', // 단승
  PLACE = 'PLACE', // 연승
  SHOW = 'SHOW', // 복승
  EXACTA = 'EXACTA', // 쌍승
  QUINELLA = 'QUINELLA', // 복연승
  TRIFECTA_BOX = 'TRIFECTA_BOX', // 삼복승
  TRIFECTA = 'TRIFECTA', // 삼쌍승
}

// 베팅 인터페이스
interface Bet {
  betType: BetType;
  raceId: string;
  selections: number[]; // 선택한 말 번호
  amount: number; // 베팅 금액
  combinations?: number; // 조합 수
}

// 배당 계산
function calculatePayout(betType: BetType, stake: number, odds: number): number {
  return stake * odds;
}
```
