## 🏇 주요 경마 용어

### 승식 (Bet Types)

| 용어   | 영문     | 설명                      |
| ------ | -------- | ------------------------- |
| 단승식 | Win      | 1마리가 1등               |
| 복승식 | Place    | 1마리가 1~3등             |
| 연승식 | Quinella | 2마리가 1~2등 (순서 무관) |
| 쌍승식 | Exacta   | 2마리가 1~2등 (순서 O)    |

### 승률 (Win Rates)

| 용어   | 설명            |
| ------ | --------------- |
| 단승률 | 1위 입상률      |
| 복승률 | 2위 이내 입상률 |
| 연승률 | 3위 이내 입상률 |
