## 연령 등급

**권장 등급: 17+ (Frequent/Intense Gambling)**

**사유 및 IARC 설문 전략:**
- Apple은 경마 관련 앱에 보수적. 도박 관련 주제는 기본적으로 17+ 등급 부여.
- "Simulated Gambling" — 앱 내 가상 베팅 기능 없음 → None 선택.
- "Real Money Gambling" — 앱 내 현금 베팅 없음 → None 선택.
- 경마를 주제로 하는 정보 서비스이므로 17+ 등급은 적절한 수준.
- 설문 시 도박 관련 항목 전부 "None" 선택 후 실제 등급 결과를 확인할 것.

> **참고:** 일부 정보 서비스 앱은 12+ 또는 17+로 심사 통과 사례 있음.
> 심사 과정에서 Apple 리뷰어가 추가 문의할 경우를 대비해,
> "This app provides information and analysis services for horse racing events.
> No betting, wagering, or gambling features are included." 설명 문구를 준비할 것.

## 인앱 구매 (In-App Purchases) 등록

> **주의:** TossPayments WebView 결제는 Apple 인앱결제(IAP) 아님.
> WebView 안에서 외부 웹 결제를 처리하는 구조이므로 App Store 수수료 미적용.
>
> 단, Apple은 디지털 콘텐츠 구독에 자사 IAP를 의무화하는 정책을 강화하고 있음.
> 심사 리젝 시 대응 전략:
> - 방어 1: "구독은 웹 서비스 결제이며 앱은 뷰어 역할" 논리 적용
> - 방어 2: 앱 내 구독 결제 UI를 제거하고 외부 웹(Safari)으로 유도하는 방식 검토
> - 방어 3: Apple IAP 연동 추가 (30% 수수료 발생 — 비용 대비 효과 검토 필요)

## Apple 심사 주의사항

1. **Guideline 4.3 (Spam):** 단순 WebView 앱은 리젝 리스크. Native 기능(푸시 알림, 네이티브 인증) 필수.
   - OddsCast는 React Native + 네이티브 푸시 알림 구현 → 해당 리스크 낮음.

2. **Guideline 3.1 (Payments):** 디지털 콘텐츠 구독을 외부 결제로 처리하는 경우 심사 주의.
   - 구독 플랜 결제 UI를 앱 내에서 직접 노출하지 않고 웹으로 유도하는 방식 검토.

3. **Guideline 5.3 (Gaming, Gambling):** 도박 앱은 지역 면허 필요.
   - 본 앱은 정보 서비스 → 앱 설명에 "정보 서비스" 명확히 기재.
   - 심사 노트(Review Notes)에 서비스 성격 설명 포함 필수.

4. **심사 노트 예시 (Review Notes 입력용):**

```
This app is an AI-powered horse racing analysis and information service.

Key points for your review:
- NO in-app betting or wagering functionality
- NO real money gambling features
- NO virtual currency with real money value
- All content is horse racing data analysis and AI-generated predictions for reference purposes only
- Payment is for an information subscription service (similar to stock market analysis newsletters)
- Actual horse racing bets can only be placed at Korea Racing Authority (KRA) official channels, outside this app

The app uses React Native with a WebView component to display our web service.
Native features include: push notifications, native authentication bridge.

This app is intended for adult users (18+) interested in horse racing information.
```
