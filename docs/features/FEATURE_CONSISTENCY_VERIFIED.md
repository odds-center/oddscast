## 🎓 핵심 개념 (통일됨)

### 1. AI 예측권 = 정보 열람권

```
예측권 사용 → AI 예측 정보 보기
정보 참고 → 외부에서 마권 구매
```

### 2. 구독료 = 정보 서비스 요금

```
월 19,800원 = AI 예측 정보 구독료
≠ 베팅 금액
≠ 충전 금액
```

### 3. 베팅 기록 = 기록 관리 도구

```
외부 구매 마권 → 앱에 기록
자동 결과 확인 → 통계 제공
≠ 앱 내 구매
```
